# Oculus Au Naturel

A fork of [Oculus](https://github.com/Asek3/Oculus) (itself a Forge port of [Iris Shaders](https://github.com/IrisShaders/Iris)), customized for use with the Au Naturel modpack.

## Fork Information

This repository is a customized fork maintained for specific modpack compatibility. It includes modifications for improved integration with Distant Horizons and other mods in the Au Naturel modpack.

### Original Projects

- **Iris Shaders**: The original Fabric shader mod - [IrisShaders/Iris](https://github.com/IrisShaders/Iris)
- **Oculus**: The Forge port by Asek3 - [Asek3/Oculus](https://github.com/Asek3/Oculus)

## Disclaimer

Oculus is not and never will be compatible with OptiFine.

## Dependencies

Oculus requires [Embeddium](https://www.curseforge.com/minecraft/mc-mods/embeddium) (or a compatible Sodium port for Forge).

## Features

* **Performance**: Fully utilizes your graphics card when paired with optimization mods like Embeddium.

* **Mod compatibility**: Makes a best effort to be compatible with modded environments.

* **Shader pack support**: All existing ShadersMod / OptiFine shader packs should work without modifications.

* **Distant Horizons integration**: Enhanced compatibility with the Distant Horizons LOD mod.
